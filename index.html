Что такое R?

  • R - это язык программирования и рабочая среда с открытым исходным ходом.
  • Изначально создавался для статистических вычислений.
  • Но сегодня умеет все (или почти все): 20928 открытых пакетов для работы.
  • Кросс-платформенный.
  • Своя среда разработки.

10 причин полюбить R

  1. мощные инструменты для работы с графикой
  2. полный набор методов компьютерного анализа текста
  3. импорт из файлов любого формата
  4. лучшая на свете среда разработки (и облако)
  5. средства для презентации и публикации исследований
  6. дружелюбное коммьюнити и прекрасные учебники (в открытом доступе)
  7. инструменты для статистического анализа
  8. возможности для машинного и глубокого обучения
  9. семейство пакетов tidyverse, которые умеют все
  10. интерактивные приложения shiny
